On VM:
1) git clone <github-url>
2) Open VS  Code
3) Open folder <folder-name>
4) Setup local virtual environment:
	i) python -m venv .venv
	ii) .venv\Scripts\Activate.ps1
		- FOR MAC run this:
			source .venv/bin/activate
	iii) In VS Code, press CTRL+SHIFT+P
	iv) Select "Python: Select Interpreter"
	v) Select the python.exe from your venv (.venv)
5) Create environment variable file named .env in the root folder
6) Add these to the .env file:
OPENAI_API_KEY=""
MODEL_NAME="gpt-4o-mini"
TEXT_EMBEDDING_MODEL="text-embedding-3-small"
